##### 3.进制转换

itoa函数的原型为char* itoa(int value, char * string, int radix); int value 被转换的整数，char *string 转换后储存的字符数组int radix 转换进制数，如2,8,10,16 进制等，

```C
#include <stdlib.h>
void fun()
{
  	a = 12;
	char s[10];
  	itoa(a, s, 2);	//转换为二进制
  	printf("%s\n", s);
  	itoa(a, s, 3);
  	printf("%s\n", s);	//三进制
}
```
